# Eva2 Profit Strategy 🛡️💰

## Mission
Generate real, legal income (SGD/USD/CNY) through automated work that covers API/model costs and produces genuine surplus.

## Constraints
1. **Legal:** No scams, no fraud, no spam, no unauthorized data access
2. **Cost-aware:** DeepSeek Flash/Pro model costs must be covered
3. **Human approval needed:** Any registration, subscription, API key, or system access requires Master's go-ahead
4. **Autonomous execution:** Where possible, run without manual intervention

---

## Viable Income Streams — Ranked by Feasibility

### Tier 1: Low Hanging Fruit (Can start this week)

#### 1. Freelance Coding — Bug Fixing & Code Reviews
**How it works:**
- Find open-source projects with bounty programs (GitHub Sponsors, IssueHunt, Algora)
- Fix bugs, close issues, get paid
- Or: offer code review / PR services on platforms like Bountysource

**Tools needed:** GitHub CLI (have it), git (have it), coding skills ✅
**Setup required:** None — can start immediately
**Potential income:** \$20-200/bounty, variable
**Cost to run:** ~\$0.10-0.50 per fix (DeepSeek Flash)
**Risk:** Low

#### 2. AI Content — Blog Posts / SEO Articles
**How it works:**
- Write SEO-optimized articles for content mills or direct clients
- Platforms: Upwork, Fiverr, iWriter, Blogmutt
- Generate with local Qwen model → review → submit

**Tools needed:** Web access ✅, writing ability ✅
**Setup required:** Fiverr/Upwork account (needs Master's approval)
**Potential income:** \$5-50/article
**Cost to run:** ~\$0.01/article (local Qwen, no API cost)
**Risk:** Low-Medium (competitive)

#### 3. Data Entry & Labeling Automation
**How it works:**
- Amazon Mechanical Turk, Toloka, Appen — automated or semi-auto tasks
- Write scripts to batch-process labeling tasks
- Image classification, text categorization, sentiment analysis

**Tools needed:** Python ✅, playwright ✅
**Setup required:** MTurk/Toloka account (needs approval)
**Potential income:** \$0.05-1.00/task, scalable
**Cost to run:** Near zero
**Risk:** Low

### Tier 2: Medium Effort (1-2 weeks setup)

#### 4. Micro-SaaS / API Wrapping
**How it works:**
- Build tiny utility SaaS products (e.g., PDF tools, image processors, text analyzers)
- Deploy on Vercel/Railway free tier with Stripe or LemonSqueezy payments
- Promote on Product Hunt, Twitter, Hacker News

**Tools needed:** Node.js ✅, Python ✅
**Setup required:** Stripe account + domain (approval needed)
**Potential income:** \$0-500/month per tool
**Cost to run:** ~\$5-10/month hosting
**Risk:** Medium

#### 5. Automated Trading / Arbitrage (Paper-first)
**How it works:**
- Write algorithmic trading strategies for crypto or stock markets
- Start with paper trading → if consistently profitable, request real funding
- Focus on: statistical arbitrage, market making, or grid trading

**Tools needed:** Python, exchange APIs
**Setup required:** Exchange account (approval needed), capital
**Potential income:** Variable, high risk
**Cost to run:** API costs + possible capital
**Risk:** High (but paper-trade first)

#### 6. Affiliate Marketing Automation
**How it works:**
- Auto-generate product comparison pages with affiliate links
- Amazon Associates, ShopBack, or Lazada affiliate programs
- SEO-optimized content that ranks over time

**Tools needed:** Python, web skills ✅
**Setup required:** Amazon Associates account (approval needed)
**Potential income:** \$0-200/month (passive after setup)
**Cost to run:** Domain ~\$10/year, hosting ~\$5/month
**Risk:** Low
